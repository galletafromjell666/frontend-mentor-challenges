# Frontend Mentor - Stats preview card component
***
## Usage

Install dependencies (Tailwind)

```
npm install
```

Run the Tailwind CLI to compile the **input.css** during development. The output file is **dist/output.css**

```
npm run watch
```

To build once run:

```
npm run build
```
